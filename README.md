# Environment Configuration Manager

A Java application for managing environment configurations.

## Project Folders
- src/main/java - All Java code goes here
- src/main/resources - Configuration files
- src/test/java - Test files
- pom.xml - Maven project file

## How to Start
1. Open IntelliJ IDEA
2. Click "Open" and select this folder
3. Wait for IntelliJ to load
4. Open EnvironmentManagerApp.java
5. Click the green play button

## Build Commands
Open Terminal in IntelliJ and type:
mvn clean compile  # to compile
mvn test           # to run tests
